PROBLEMAS DE MOVILIDAD

En problemas de movilidad, se suelen trabajar con dos o más objetos que se desplazan en distintas direcciones, a velocidades constantes o variables. Para resolver este tipo de problemas, es necesario conocer algunas fórmulas y conceptos fundamentales:

▶ VELOCIDAD:

La velocidad se define como el cambio de distancia respecto al tiempo. Si un objeto se desplaza una distancia "d" en un tiempo "t", su velocidad promedio será "d/t". La velocidad se mide en unidades de distancia por unidad de tiempo, como metros por segundo (m/s) o kilómetros por hora (km/h).

▶ TIEMPO DE ENCUENTRO:

El tiempo de encuentro es el tiempo que transcurre desde que dos objetos comienzan a moverse hasta que se encuentran en un mismo punto. Para calcular el tiempo de encuentro, se puede utilizar la siguiente fórmula:

T = d/[V₁ + V₂]

Donde "d" es la distancia de separación entre ambos móviles, "V₁" y "V₂" son las velocidades de los móviles.

▶ TIEMPO DE ALCANCE:

El tiempo de alcance es el tiempo que transcurre desde que un objeto comienza a moverse hasta que alcanza a otro objeto que ya se encuentra en movimiento. Para calcular el tiempo de alcance, se puede utilizar la siguiente fórmula:

T = d /[V₁ - V₂]

Donde "d" es la distancia de separación entre ambos móviles, "V₁" es la velocidad del móvil más rápido y "V₂" es la velocidad del móvil más lento.

▶ MÓVIL PASANDO POR UN TÚNEL:

En este tipo de problemas, se debe calcular la velocidad de un objeto que se mueve a través de un túnel, cuya longitud se conoce. Para resolver este tipo de problemas, se puede utilizar la siguiente fórmula:

  L + X = V.T

 Donde "V" es la velocidad del móvil, "L" la longitud del móvil, "T" el Tiempo para cruzar el túnel, y "X" la longitud del túnel.
